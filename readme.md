# is-root-path

> Check if a path is the root path. Example: `/` or `C:\`


## Install

```
$ npm install is-root-path
```


## Usage

```js
const isRootPath = require('is-root-path');

isRootPath('/');
//=> true

isRootPath('/Users');
//=> false
```
